## 任务 1: 阿里云 OSS 开通 ⭐

### 步骤

1. 登录阿里云控制台
   - 网址：https://oss.console.aliyun.com

2. 创建 Bucket
   - 点击「创建 Bucket」
   - 命名：`lvzhi-assets`（或其他名称）
   - 地域：**华东 2（上海）**（与 PolarDB 同一地域，减少延迟）
   - 存储类型：标准存储
   - 读写权限：**私有**

3. 获取 Access Key
   - 进入「 Access Key 管理」
   - 创建新的 Access Key 或使用已有密钥
   - 记录 `AccessKey ID` 和 `AccessKey Secret`

### 需要记录的信息

```
OSS Endpoint: oss-cn-shanghai.aliyuncs.com（根据实际地域）
Bucket 名称: lvzhi-assets
AccessKey ID: ________________
AccessKey Secret: ________________
```

## 任务 2: OSS Bucket 权限配置

### 步骤

1. 在 OSS 控制台进入刚创建的 Bucket

2. 配置跨域规则（CORS）
   - 进入「数据安全」→「跨域设置」
   - 点击「创建规则」
   - 来源：填写你的网站域名（暂时用 `*`，备案后改为实际域名）
   - 允许 Methods：GET, POST, PUT, DELETE, HEAD, OPTIONS
   - 允许 Headers：`*`
   - 暴露 Headers：ETag, x-oss-request-id

3. 配置生命周期规则（可选）
   - 进入「数据处理」→「生命周期」
   - 建议：90天后转为归档存储

4. 配置防盗链（可选，备案后配置）
   - 进入「数据安全」→「Referer 白名单」
   - 添加你的网站域名

## 任务 3: PolarDB 环境变量配置

### 你的 PolarDB 信息（已确认）
- 地域：华东 2（上海）
- 版本：PostgreSQL 14
- 连接类型：公网地址

### 步骤

1. 在阿里云 PolarDB 控制台获取连接信息
   - 网址：https://polardb.console.aliyun.com

2. 创建数据库账号（如果没有）
   - 进入「账号管理」→「创建账号」
   - 账号名：`lvzhi`
   - 账号类型：高权限账号
   - 设置密码

3. 创建数据库
   - 进入「数据库管理」→「创建数据库」
   - 数据库名：`lvzhi`

4. 获取连接地址
   - 在「数据库管理」页面找到公网连接地址
   - 格式：`pgm-xxxxxxxxxxxx.pg.rds.aliyuncs.com`

### 环境变量配置

在 `律植项目/deploy/.env` 中填写：

```bash
# 数据库配置（使用 PolarDB）
POSTGRES_HOST=pgm-xxxxxxxxxxxx.pg.rds.aliyuncs.com
POSTGRES_PORT=5432
POSTGRES_DB=lvzhi
POSTGRES_USER=lvzhi
POSTGRES_PASSWORD=你的密码

# 完整连接字符串
DATABASE_URL=postgresql://lvzhi:你的密码@pgm-xxxxxxxxxxxx.pg.rds.aliyuncs.com:5432/lvzhi
```

- [ ] PolarDB 公网连接地址已获取

- [ ] 数据库 `lvzhi` 已创建

  - 进入「数据安全性」→「白名单设置」

## 任务 4: Supabase 用户数据导出

### 准备工作

1. 登录 Supabase 仪表板
   - 网址：https://supabase.com/dashboard

2. 进入你的项目 → Settings → API

3. 记录以下信息：
   ```
   Project URL: https://xxxx.supabase.co
   anon/public key: ________________
   service_role key: ________________
   ```

### 导出用户数据

在 Supabase SQL Editor 执行：

```sql
-- 导出所有用户（排除已删除的）
SELECT 
  id,
  email,
  phone,
  display_name,
  avatar_url,
  role,
  created_at,
  updated_at,
  email_confirmed_at IS NOT NULL as email_verified,
  phone_confirmed_at IS NOT NULL as phone_verified
FROM auth.users
WHERE deleted_at IS NULL;

-- 导出用户元数据
SELECT 
  id,
  raw_user_meta_data
FROM auth.users;

-- 导出 profiles 表（如果有）
SELECT * FROM public.profiles;
```

点击「Download as CSV」保存数据。

### 导出后需要的数据

1. 用户映射表（CSV）
   - 原 Supabase 用户 ID → 新系统用户 ID
   - 用于关联历史数据

2. 如果有 OAuth 用户
   - WeChat unionid/openid
   - 支付宝 user_id
   - 用于迁移第三方登录绑定

## 任务 8: ICP 备案申请

### 说明
在中国大陆提供互联网信息服务，必须进行 ICP 备案。这是上线中国大陆的最关键前置任务。

### 备案前提
1. 拥有国内服务器（阿里云 ECS/腾讯云 CVM）
2. 拥有域名且已完成实名认证
3. 网站内容符合国内法规要求

### 备案流程
1. 登录阿里云备案系统：https://beian.aliyun.com
2. 选择备案类型（首次备案/新增网站）
3. 填写网站信息
   - 网站名称：律植
   - 网站域名：lvzhi.com
   - 网站内容：法律智能服务平台
4. 上传资料
   - 企业/个人证件
   - 网站负责人身份证
   - 网站备案信息真实性核验单
5. 阿里云审核（1-2 工作日）
6. 管局的最终审核（7-20 工作日）

### 备案号
备案成功后，在网站底部添加备案号：

1. 登录阿里云 SSL 证书控制台
2. 选择「免费证书」
3. 创建证书
   - 填写域名：lvzhi.com, *.lvzhi.com
4. 自动 DNS 验证
5. 签发后下载证书

### 证书配置
将证书文件放置在服务器：
```
/etc/nginx/ssl/lvzhi.com.pem
/etc/nginx/ssl/lvzhi.com.key
```

### nginx 配置
```nginx
server {
    listen 443 ssl http2;
    server_name lvzhi.com;

    ssl_certificate /etc/nginx/ssl/lvzhi.com.pem;
    ssl_certificate_key /etc/nginx/ssl/lvzhi.com.key;

    # 其他配置...
}

server {
    listen 80;
    server_name lvzhi.com;
    return 301 https://$server_name$request_uri;
}
```
