```bash

```

### 2. 配置环境
```bash
cp .env.example .env.local
# 编辑 .env.local，填入��
# - 数据库连接
# - Supabase URL 和密钥
# - 其他API密钥
```

### 3. 启动开发服务器
```bash
# 前端

# 后端

```
