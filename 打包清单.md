## 八、如何使用此代码包

### 1. 环境准备
```bash
# 安装依赖
pnpm install

# 复制环境变量
cp .env.example .env.local
# 编辑 .env.local，填入数据库连接、Supabase密钥等

# 启动数据库（如需）
# 参考 docker-compose.yml

# 运行开发服务器
pnpm dev
```

### 2. 访问应用
- 前端：http://localhost:3000
- 后端API：http://localhost:3001
- 管理后台：http://localhost:3000/admin

### 3. 数据库初始化
```bash
# 运行迁移脚本
psql -U postgres -f supabase/migrations/XXX.sql
# 或使用 Supabase CLI
supabase db push
```
